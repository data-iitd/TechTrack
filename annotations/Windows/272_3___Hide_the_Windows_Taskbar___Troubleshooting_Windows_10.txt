3 - Copy and paste the following command.
Copy and paste the following command. Make sure you paste it into the "Administrator" window:
- Get-AppXPackage -AllUsers | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}

4 - Run the command.
Run the command. You may see a few errors as the command runs, which can safely be ignored.

5 - Click or tap the Start menu after the command finishes.
Click or tap the Start menu after the command finishes. You should notice that the taskbar hides as it should and stays hidden.[4]
